What is UCP?

The USB for Commerce

Universal Commerce Protocol

One standard API for all shopping:
Discover → Cart → Checkout → Pay → Order

AI agents use UCP to shop on behalf of users across any merchant, any payment provider.

  • Payments: Visa, Mastercard, Stripe, Adyen
  • Ecosystem: 20+ endorsed partners
